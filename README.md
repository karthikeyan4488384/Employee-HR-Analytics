# Employee HR Analytics Project

## Project Overview
This project analyzes employee HR data to evaluate department performance, salary distribution, retention risk, and promotion readiness using Python-based data analysis techniques.
## Project Objective
To derive actionable business insights from employee data and support HR decision-making in areas such as:
- Department strength analysis
- Salary imbalance detection
- Retention risk identification
- Promotion-ready employee evaluation

##  Dataset Description
The dataset contains employee information including:

- Department
- Salary
- Experience Years
- Performance Rating
- Join Date


## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Google Colab


##  Key Analysis Performed

- Data cleaning and missing value handling
- Salary band classification using lambda functions
- Department-wise aggregation (salary, experience, performance)
- Identification of strongest department
- Retention risk analysis
- Promotion-ready employee identification

##  Key Insights

- Sales department emerged as the strongest overall based on performance and experience metrics.
- Salary imbalance observed among high-experience employees in medium salary bands.
- No high-risk retention employees identified under strict criteria.
- Promotion-ready candidates identified using tenure and performance conditions.


##  Business Impact
This analysis demonstrates how data-driven insights can support HR strategy, compensation planning, and performance management decisions.
